# PIT Übersicht



##  Zurücksetzen



Enthält alle Server und Schienen die zu uns zurück gekommen sind und überprüft und zurückgesetzt werden müssen

Wenn das Zurücksetzen erfolgte kann der Button Zurücksetzen angeklickt werden und die Schiene oder der Server werden auf den Status Lager gesetzt



## Server und Schienen im Lager



Hier werden Server und Schienen auf geführt die Versandfertig sind. Sobald diese ein DPD Packetschein bekommen haben und ins Abhol Lager geschoben wurden. Kann der Button Versand angeklickt werden.





## Versand/Weiterleitungen/Rückholungen



Sollte eigentlich selbterklärend sein. Ist was an DPD Übergeben worden oder von DPD abgeholt worden einfach auf Ja Klicken. Ab da haben wir eh kein Einfluß mehr was passiert.
